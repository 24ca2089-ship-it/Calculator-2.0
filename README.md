# 🎓 BCA 4th Semester — Grade Calculator

A clean, interactive **JavaScript Grade Calculator** built for BCA 4th Semester students. Enter subject names and marks to instantly get your percentage, CGPA, grades, and result division.

---

## 🌟 Features

- ✅ Pre-filled with **BCA 4th Sem subjects** (Computer Networks, OS, Java, DBMS, etc.)
- ➕ Add / remove subjects dynamically
- 📊 Calculates **Percentage**, **CGPA (out of 10)**, and **Grade** per subject
- 🏅 Overall grade badge: O / A+ / A / B+ / B / C / F
- 📋 Detailed subject breakdown
- 🖨️ Print / Save as PDF support
- 💡 Live grade preview as you type
- 📱 Fully responsive (mobile-friendly)
- 🎨 Modern dark UI — no frameworks needed!

```bash
cd bca-grade-calculator
open index.html
```

## 📊 Grading System

| Percentage | Grade | Grade Points |
|------------|-------|--------------|
| ≥ 90%      | O     | 10           |
| ≥ 80%      | A+    | 9            |
| ≥ 70%      | A     | 8            |
| ≥ 60%      | B+    | 7            |
| ≥ 50%      | B     | 6            |
| ≥ 40%      | C     | 5            |
| < 40%      | F     | 0            |

---

## 🏆 Division System

| Percentage | Result        |
|------------|---------------|
| ≥ 75%      | Distinction   |
| ≥ 60%      | First Division |
| ≥ 45%      | Second Division |
| ≥ 33%      | Third Division |
| < 33%      | Fail          |
